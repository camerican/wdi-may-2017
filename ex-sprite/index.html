The above animation is made possible through use of a sprite sheet. This particular sprite sheet contains 10 drawings of a character in a running motion. We only want to display one at a time, which we can do through ensuring that the div size is only enough to fit a single drawing from the grid and then sliding the image around via background-position to create for the other frames.
